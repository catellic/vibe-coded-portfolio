## 📝 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/catellic/vibe-coded-portfolio.git
   cd vibe-coded-portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run locally:**
   ```bash
   npm run dev
   ```
4. **Open in your browser:**  
   Visit `http://localhost:3000`
